# 打 Victor!

一個有趣的網頁遊戲,讓你發洩對 Victor 的不滿!

## 遊戲特色

- 點擊不同部位攻擊 Victor:
  - 打眼鏡 👓
  - 扯頭髮 ✂️
  - 戳臉蛋 👆
- 每次攻擊都有動畫效果和自定義音效:
  - 「哎也」和「好痛」隨機播放
  - 憤怒模式時音效更加強烈
- 打到 10 下會觸發憤怒模式!
- 支援電腦和手機操作

## 如何開始

1. 下載或克隆此專案
2. 用瀏覽器打開 `index.html`
3. 開始攻擊 Victor!

## 技術實現

- HTML5 + CSS3 + JavaScript
- SVG 動畫和表情系統
- 響應式設計適配各種設備
- 自定義音效系統
- 進度條和計分系統
